# RedSocial App 🌐

Aplicación web de red social construida con **HTML, CSS (W3.CSS) y JavaScript vanilla**.

## Páginas

| Página | Descripción |
|---|---|
| `login.html` | Inicio de sesión |
| `registro.html` | Crear cuenta nueva |
| `plantilla-RedSocial.html` | Feed principal |
| `perfil.html` | Perfil del usuario |
| `chat.html` | Mensajes / Chat |
| `grupos.html` | Grupos |
| `configuracion.html` | Configuración de cuenta |

## Características

- ✅ Rutas restringidas con `sessionStorage` (redirige al login si no hay sesión)
- ✅ Login y registro con manejo de sesión
- ✅ Cerrar sesión desde cualquier página
- ✅ Navbar responsive con menú móvil
